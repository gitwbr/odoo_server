- [x] 2.1 建立概覽統計服務，集中封裝 read_group/search_count 查詢。
- [x] 2.2 大圖訂單 KPI：今日/選定期間訂單數、狀態數、類別數、未稅金額、含稅金額、總才數。
- [x] 2.3 大圖訂單出貨結果：交期風險、已轉/未轉 S、母單交貨方式排行保留在大圖訂單區塊。
- [x] 2.4 工單狀況 KPI：B/C/G/T 需求、已轉、待轉、待完成、已完成與狀態分布。
- [x] 2.5 財務收付第一版：期間應收/應付、收付差額、應收應付對比、趨勢與客戶/供應商排行；不顯示未收/未付，因現有系統未維護實際收付款狀態。
- [x] 2.6 加入對應清單跳轉，點擊 KPI 可進入過濾後列表。
- [ ] 2.7 出貨明細延伸資料：在大圖訂單出貨分析下，依 `dtsc.deliveryorder` 與 `dtsc.deliveryorderline.make_orderid` 統計 S 單來源、出貨才數，以及 S 關聯 A/B/C/G/T 等來源分布。
